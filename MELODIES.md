# Available Melodies

This document lists all 64 predefined melodies from JR East departure melodies available in hassha.

## Usage

Use any melody ID in your `.hassha/config.toml`:

```toml
[hooks.Stop]
melody = "JY-Shibuya"
volume = 0.8
```

## Custom Audio

You can also use custom audio sources:

```toml
# URL
[hooks.Stop]
melody = "https://example.com/sound.mp3"

# Local file
[hooks.SessionStart]
melody = "/path/to/startup.mp3"
```

## Testing Melodies

Play any melody directly from the command line:

```bash
hassha play JY-Shibuya
hassha play JK-Akihabara --volume 0.5
hassha play NEX-Shinjuku
```
